## About The Project
This is a project for learning the slicing part of a given template. After slicing it can edited dynamically.


---
## Features: 

### Itally Food :
* Home page design
* Blog page design
* Menu is added dynamically 
* Header & Footer is added dynamically
* Staff page design
* Sidebar is added dynamically
